# 房小团-业务组件库

> 常用小程序uniapp业务UI组件

## 安装

```
npm i @hjtui/mpui --save

# 需要配合scss使用
npm i sass sass-loader -D
```

## 使用

```js
// 引入
import BusiBanner from '@hjtui/mpui/components/组件文件夹/组件.vue';
```
